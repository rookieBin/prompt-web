# PromptHub Assistant Chrome 插件

一个Chrome浏览器插件，让你能够快速访问和复制收藏的AI提示词。

## 功能特性

- 🔍 **快速搜索**: 在插件弹窗中搜索你收藏的提示词
- 📋 **一键复制**: 点击即可复制提示词内容到剪贴板
- 🎨 **美观界面**: 与PromptHub网站保持一致的设计风格
- 🔄 **数据同步**: 与主网站的收藏数据保持同步
- 🌐 **全网可用**: 在任何网站都能使用你的提示词库

## 安装方法

### 开发环境安装

1. 打开Chrome浏览器，访问 `chrome://extensions/`
2. 开启右上角的"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择 `chrome-extension` 文件夹
5. 插件安装完成，可以在工具栏看到PromptHub图标

### 打包安装

1. 在`chrome-extension`目录中，选中所有文件
2. 压缩为 `.zip` 文件
3. 在Chrome扩展程序页面，点击"打包扩展程序"
4. 选择压缩文件进行安装

## 使用方法

### 基本操作

1. **打开插件**: 点击工具栏中的PromptHub图标
2. **搜索提示词**: 在搜索框中输入关键词进行搜索
3. **复制提示词**: 点击任意提示词卡片或"复制"按钮
4. **刷新数据**: 点击底部"刷新数据"按钮同步最新收藏
5. **打开网站**: 点击"打开网站"直接访问PromptHub

### 快捷键

- `Ctrl+Shift+P`: 快速打开插件（需在PromptHub网站上）

### 网站集成

当你访问PromptHub网站时，插件会自动：
- 检测页面上的提示词操作
- 同步新收藏的提示词到插件
- 显示操作反馈通知

## 技术实现

### 数据存储
- 使用Chrome Storage API存储收藏的提示词
- 支持多设备同步（需登录Chrome账户）
- 本地缓存确保离线访问

### 权限说明
- `storage`: 存储用户收藏的提示词
- `clipboardWrite`: 复制内容到剪贴板
- `<all_urls>`: 在所有网站注入content script

### 架构设计
- **Popup**: 用户交互界面，显示提示词列表
- **Background**: 后台服务，处理数据存储和同步
- **Content Script**: 与网页交互，检测PromptHub网站操作

## 开发说明

### 本地开发

1. 修改代码后，在Chrome扩展程序页面点击刷新按钮
2. 对于content script的修改，需要刷新对应的网页
3. 查看控制台日志进行调试

### 调试方法

- **Popup调试**: 右键点击插件图标 → 检查弹出内容
- **Background调试**: 扩展程序页面 → 服务工作进程 → 检查视图
- **Content Script调试**: 在网页上按F12，查看控制台

### 自定义配置

可以通过修改以下文件来自定义插件：

- `popup.js`: 修改模拟数据、添加新功能
- `styles.css`: 调整界面样式和主题
- `background.js`: 修改数据同步逻辑
- `content.js`: 调整网站集成功能

## 与主项目集成

插件设计时考虑了与主PromptHub项目的紧密集成：

1. **共享数据结构**: 使用相同的提示词数据格式
2. **样式一致性**: 复用主项目的Element Plus设计语言
3. **API兼容**: 预留了与后端API的对接接口
4. **状态同步**: 支持与网站收藏状态的实时同步
